# Subash Nepal — Portfolio

Personal portfolio website for **Subash Nepal** — Video Editor, 3D Artist & Web Designer from Bhairahawa, Nepal.

🔗 **Live site:** [subashnepal.com](https://subashnepal.com)

---

## Tech Stack

- **Framework:** [Next.js 16](https://nextjs.org/) (App Router)
- **Language:** TypeScript
- **Styling:** Vanilla CSS (Neobrutalism design system)
- **Fonts:** Space Grotesk + Inter (Google Fonts)
- **Deployment:** Vercel

## Pages

| Route | Description |
|---|---|
| `/` | Hero, services preview, featured projects, CTA |
| `/about` | Bio, skills, timeline |
| `/services` | Full services listing |
| `/projects` | Project portfolio with links |
| `/contact` | Contact form (Formspree) + social links |

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

This project is deployed via **Vercel** connected to this GitHub repository.  
Every push to `main` triggers an automatic redeploy.
